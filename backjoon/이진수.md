# Backjoon : 이진수

## Problem URL
https://www.acmicpc.net/problem/3460

<br/>

## 🚩 Solution
```js
const input =  require('fs').readFileSync('/dev/stdin').toString().trim().split('\n');

for(let i =0; i<Number(input[0]); i++) {
    
    let num = Number(input[i+1]);
    let result = [];
    let cnt = 0;

    while(num){
        if(num % 2 === 1){
            result.push(cnt);
        }
        cnt += 1;
        num = parseInt(num/2);
    }
    console.log(result.join(' '));
}
```
